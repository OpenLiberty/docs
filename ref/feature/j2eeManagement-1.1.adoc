// end::description[]
// tag::enable[]
== Enabling this feature
To enable the J2EE Management 1.1 feature, add the following element declaration inside the featureManager element in your server.xml file:


----
<feature>j2eeManagement-1.1</feature>
----
// end::enable[]
// tag::config[]
